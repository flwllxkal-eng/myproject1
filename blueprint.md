# Project Blueprint

## Overview

This project is a sophisticated and user-friendly password generator web application. It is designed not only to provide a functional tool but also to educate users on the principles of strong password hygiene and online security. The goal is to create a content-rich, visually appealing page that meets the quality standards for platforms like Google AdSense.

## Implemented Features & Design

*   **Advanced Password Generation:** Generates a random password (8-12 characters) that is guaranteed to include at least one uppercase letter, one lowercase letter, one number, and one special character. This ensures compliance with the most common and secure password policies.

*   **Modern & Secure UI:**
    *   **Dark Mode Theme:** A visually comfortable and modern dark theme reduces eye strain.
    *   **Responsive Design:** The layout is fully responsive and optimized for both desktop and mobile devices.
    *   **Interactive Elements:** Buttons and input fields have clear, interactive feedback (hover effects, shadows) to enhance usability.
    *   **Professional Typography:** Uses a combination of "Noto Sans KR" and "Roboto" for excellent readability and a clean aesthetic.

*   **Rich Educational Content:**
    *   **"Golden Rules" Section:** Provides a detailed explanation of what makes a password strong (length, complexity, uniqueness) to educate the user.
    *   **Tool Justification Section:** Explains the purpose and security benefits of using a random password generator. It emphasizes that the tool operates entirely on the client-side, ensuring user privacy and data security.

*   **Secure Copy-to-Clipboard:** Utilizes the modern `navigator.clipboard.writeText()` API for a secure and seamless way for users to copy the generated password without exposing it.
