
# Project Blueprint

## Overview

This project is a password generator web application. It allows users to generate a random password and copy it to their clipboard.

## Implemented Features

*   **Password Generation:** Generates a random password between 8 and 12 characters, including uppercase letters, lowercase letters, numbers, and special characters. **It is now guaranteed that at least one of each character type is included.**
*   **Copy to Clipboard:** Allows the user to copy the generated password to their clipboard. The copy function has been updated to use the modern Clipboard API for better security and user experience.

## Current Plan

*   **HTML:** Create the basic structure of the password generator, including a display for the password, a "Generate Password" button, and a "Copy to Clipboard" button. (Completed)
*   **CSS:** Style the elements to be visually appealing and user-friendly. (Completed)
*   **JavaScript:** Implement the logic for generating the password and copying it to the clipboard. (Updated with guaranteed character types and modern Clipboard API)
