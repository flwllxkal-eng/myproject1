How to Use

This calculator functions like a standard scientific calculator. It respects the order of operations (PEMDAS/BODMAS).

  • AC (All Clear): Clears all input and resets the calculator.
  • ( ) : Use parentheses to group operations and control the order of evaluation.
  • sin, cos, tan: Calculates the sine, cosine, or tangent of a number (assumed to be in radians). Example: sin(3.14)
  • log: Calculates the base-10 logarithm. Example: log(100)
  • Standard Operations: Use +, -, ×, ÷ for basic arithmetic.

Rules & Features

Order of Operations: The calculator automatically follows the standard order of operations (Parentheses, Exponents, Multiplication/Division, Addition/Subtraction). For example, 2+3*4 will correctly result in 14.

Input: Click the buttons to build your mathematical expression. The expression is evaluated when you press the = button.

Error Handling: Invalid expressions like 2++3 or mismatched parentheses will result in an "Error" message. Press 'AC' to start over.
