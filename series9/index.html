How to Use

  • Deg/Rad: Toggles angle units for trig functions (sin, cos, tan).
  • Memory (M+/M-/MR/MC): Store, subtract from, add to, recall, and clear memory.
  • Advanced Functions: Includes `x²` (square), `xʸ` (power), `√` (sqrt), `x!` (factorial), `log`, `ln`, `π`, and `e`.
  • EXP: Use for scientific notation (e.g., `3.2e+5`).
