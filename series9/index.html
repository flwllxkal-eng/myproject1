How to Use & Features

This calculator supports both mouse clicks and keyboard input for all its functionalities.

  • Basic Operations: Use 0-9, +, -, *, /, and . for standard arithmetic. Press Enter to calculate or =.
  • Clear & Delete: AC (or Escape) clears the entire expression. DEL (or Backspace) removes the last character.
  • Parentheses (): Group operations to control the order of calculation.
  • Constants (π, e): Use π for Pi (3.14159...) and e for Euler's number (2.71828...).
  • Trigonometry (sin, cos, tan): Calculates the trigonometric function of a value. Assumes input in radians (e.g., sin(π/2)).
  • Logarithms (log, ln): log is the base-10 logarithm, and ln is the natural (base-e) logarithm.
  • Powers & Roots:
    • x²: Squares the current number (e.g., (5)^2).
    • xʸ: Raises a number to a power (e.g., 3^4). Use the ^ character from keyboard.
    • √: Calculates the square root (e.g., √(9)).
  • Factorial x!: Calculates the factorial of a non-negative integer (e.g., fact(5) which is 120).
  • Scientific Notation EXP: Use EXP to enter numbers in scientific notation (e.g., 3.2EXP5 is 3.2 x 105).
